In this step, you will create a dashboard directly in the Edge UI and add widgets to visualize the telemetry and alarms from `sensor-01`.

---

## Step 5.1 Create a dashboard

<Steps>
1. In the Edge UI (`http://localhost:8080`), go to **Dashboards** and click **+** &#8702; **Create new dashboard**.
2. Enter a name, like `Edge Dashboard`, and click **Add**. The dashboard opens in edit mode.
</Steps>

## Step 5.2 Add a time series chart

<Steps>
1. Click **Add widget**.
2. Select **Charts** &#8702; **Time series chart**.
3. Set `sensor-01` as the entity and `temperature` as the series key.
4. Click **Add**.
</Steps>

The chart displays temperature readings over time as new telemetry arrives.

---

## Step 5.3 Add an alarms table

<Steps>
1. Click **Add widget**.
2. Select **Alarm widgets** &#8702; **Alarms table**.
3. Set `sensor-01` as the entity.
4. Click **Add**.
</Steps>

The alarms table shows all alarms associated with `sensor-01`, including the **High temperature** alarm from the previous step.

---

## Step 5.4 Save and verify

<Steps>
1. Click **Save** in the top-right corner.
2. Send a high-temperature reading to confirm the chart updates and the alarm appears in the table:

   ```bash
   curl -X POST http://localhost:8080/api/v1/YOUR_ACCESS_TOKEN/telemetry \
     -H "Content-Type: application/json" \
     -d '{"temperature": 36}'
   ```
</Steps>
