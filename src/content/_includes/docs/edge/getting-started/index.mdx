<Lede>After completing this guide, you will have ThingsBoard Edge running on your local machine, connected to your ThingsBoard account, and ready to process device data.</Lede>

By completing this guide, you will learn how to:

- Install ThingsBoard Edge and connect it to a server
- Add a device and publish telemetry
- Filter telemetry with a rule chain node
- Set up an alarm rule
- Create and configure real-time dashboards

:::tip
Use the checkboxes at the bottom of each page to track your progress.
All data is stored in your browser and is not sent to ThingsBoard.
:::

## Prerequisites

Before starting, you need:

- An active ThingsBoard account with Edge functionality enabled (<DocLink product={props.product === Products.EDGE_PE ? Products.PE : Products.CE} path="installation" target="_blank">self-hosted</DocLink>{props.product === Products.EDGE_PE && <> or <a href="/docs/pe/installation/?installationType=saas" target="_blank">ThingsBoard Cloud</a></>}).
- A Linux-based machine or Docker environment (minimum 1 GB RAM).
- Port `7070` open between the Edge host and your ThingsBoard instance
