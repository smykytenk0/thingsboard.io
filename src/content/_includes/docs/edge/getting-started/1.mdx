## Step 1.1 Provision Edge

Before installing Edge software, create an Edge instance record in your ThingsBoard account. This generates the credentials Edge uses to authenticate.

<Steps>
1. Log in to your ThingsBoard account as a Tenant Administrator.

2. Go to **Edge Management → Instances** and click **Add Edge** (+).

3. Enter a name for the Edge instance (like `factory-edge-01`) and click **Add**.

4. Open the Edge instance and note the **Edge key** and **Edge secret**. You will need them in the next step.
</Steps>

## Step 1.2 Install ThingsBoard Edge

Run Edge using Docker.

<Steps>
1. Create a working directory and a `docker-compose.yml` file:

   {props.product === Products.EDGE && (
     <Code code={ceDockerCompose} lang="yml" title="docker-compose.yml" />
   )}
   {props.product === Products.EDGE_PE && (
     <Code code={peDockerCompose} lang="yml" title="docker-compose.yml" />
   )}

2. Replace the placeholder values:

   | Variable | Value |
   |---|---|
   | `CLOUD_ROUTING_KEY` | Edge key from Step 1.1 |
   | `CLOUD_ROUTING_SECRET` | Edge secret from Step 1.1 |
   | `CLOUD_RPC_HOST` | Hostname or IP of your ThingsBoard instance (e.g. `192.168.1.1`) |

3. Start Edge:

   ```bash
   docker compose up -d && docker compose logs -f mytbedge
   ```

4. Open `http://localhost:8080` in a browser. Log in with the default Tenant Administrator credentials (`tenant@thingsboard.org` / `tenant`). Change the password when prompted.
</Steps>

<Aside type="note">
The Edge UI runs on port `8080` by default. The MQTT broker for devices runs on port `1883`. CoAP runs on ports `5683–5688`.
</Aside>

---

## Step 1.3 Verify the connection

After Edge starts, it connects to your ThingsBoard account over gRPC and begins synchronizing.

<Steps>
1. Go back to your ThingsBoard account.

2. Go to **Edge Management → Instances** and open the Edge instance you created.

3. Check the **Status** field. It changes to **Connected** within a few seconds after Edge starts.
</Steps>

If the status stays **Disconnected**, check that `CLOUD_RPC_HOST` resolves from the Edge host and that port `7070` is reachable.
