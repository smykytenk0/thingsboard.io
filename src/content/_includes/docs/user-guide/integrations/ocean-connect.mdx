Huawei OceanConnect Integration allows you to stream data from the OceanConnect platform to ThingsBoard and convert binary device payloads to the ThingsBoard format.

<DocImage src="/src/assets/images/user-guide/integrations/ocean-connect/ocean-connect-integration.svg" alt="OceanConnect integration overview diagram" />

## Add OceanConnect integration

<Steps>
1. Go to **Integration center &#8702; Integrations** and click **+ Add integration**.
2. Select type **OceanConnect**, enter a name, and complete the integration wizard.
3. Copy the auto-generated **HTTP endpoint URL** from the connection settings — you will need it in the next step.
</Steps>

<DocImage src="/src/assets/images/user-guide/integrations/ocean-connect/ocean-connect-integration-setup.png" alt="OceanConnect integration — HTTP endpoint URL highlighted in the connection settings" />

## Configure OceanConnect

Register the copied HTTP endpoint URL in your OceanConnect account. Refer to the official <a href="https://support.huawei.com/enterprise/en/cloud-computing/oceanconnect-platform-pid-22559503" target="_blank" rel="noopener noreferrer">OceanConnect documentation</a> for instructions on how to provision the endpoint URL.
