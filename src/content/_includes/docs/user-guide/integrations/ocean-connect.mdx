<a href="https://e.huawei.com/en/products/cloud-computing-dc/iot" target="_blank" rel="noopener noreferrer">Huawei OceanConnect</a> is an IoT connectivity management platform that collects and manages data from millions of IoT devices.
The ThingsBoard OceanConnect integration receives device data pushed from OceanConnect via HTTP, decodes the payload using an uplink converter, and creates devices with telemetry and attributes in ThingsBoard automatically.

<DocImage src="/src/assets/images/user-guide/integrations/ocean-connect/ocean-connect-integration.svg" alt="OceanConnect integration overview diagram" />

## Add OceanConnect integration

<Steps>
  <ol>
    <li>Go to <strong>Integrations center &#8702; Integrations</strong> and click <strong>+ Add integration</strong>.</li>
    <li>
      <span><strong>Basic settings:</strong></span>
      <ul>
        <li>Set <strong>Integration type</strong> to <strong>OceanConnect</strong>.</li>
        <li><strong>Enable integration</strong> and <strong>Allow create devices or assets</strong> are on by default.</li>
        <li>Click <strong>Next</strong>.</li>
      </ul>
    </li>
    <li>
      <span><strong>Uplink data converter:</strong></span>
      <ul>
        <li>Choose <strong>Create new</strong> and enter a converter name.</li>
        <li>The default TBEL decoder parses the incoming OceanConnect payload as JSON. Adjust it to match your device's payload structure if needed.</li>
        <li>Click <strong>Next</strong>.</li>
      </ul>
    </li>
    <li>
      <span><strong>Connection:</strong></span>
      <ul>
        <li>Specify your <strong>Base URL</strong> (e.g. <code>https://thingsboard.cloud</code>).</li>
        <li>Copy the auto-generated <strong>HTTP endpoint URL</strong> — you will register it in OceanConnect as the push destination.</li>
      </ul>
    </li>
    <li>Click <strong>Add</strong>.</li>
  </ol>
</Steps>

#### Connection settings

<h4>Base URL</h4>

The base address of your ThingsBoard server, used to construct the HTTP endpoint URL. For ThingsBoard Cloud, use `https://thingsboard.cloud`.

<h4>HTTP endpoint URL</h4>

Auto-generated endpoint for this integration. Copy this URL and register it in OceanConnect as the subscription push address so OceanConnect forwards device uplink messages to ThingsBoard.

<h4>Execute remotely</h4>

When enabled, ThingsBoard generates an **Integration key** and **Integration secret** that allow the integration to run as a separate process outside the ThingsBoard cluster — useful when the integration must reach services not directly accessible from the ThingsBoard server.

<h4>Advanced settings</h4>

- **Description** — optional text description for this integration
- **Metadata** — key-value pairs injected into every uplink message as `integrationMetadata` in converter scripts

## Configure OceanConnect

In your OceanConnect account, create a **Subscription** (push notification rule) that targets the ThingsBoard HTTP endpoint URL copied in the previous step. OceanConnect will POST device data to this URL each time a device reports.

<Aside type="note">
  The exact steps for creating a subscription depend on your OceanConnect version and account type. Refer to the <a href="https://support.huawei.com/enterprise/en/cloud-computing/oceanconnect-platform-pid-22559503" target="_blank" rel="noopener noreferrer">official OceanConnect documentation</a> for current instructions.
</Aside>
