KPN Things is an IoT platform provided by KPN, a telecommunications company.
It allows users to connect and manage IoT devices, facilitating communication and data exchange between devices and the cloud.
After integrating KPN Things with ThingsBoard, you can connect, manage, communicate, process, and visualize data from your devices in the ThingsBoard IoT platform.

## ThingsBoard setup

### Create the KPN Things integration

<Steps>
  <ol start="1">
    <li>Go to <strong>Integration center &#8702; Integrations</strong> and click <strong>+ Add integration</strong>.</li>
    <li>
      <span><strong>Basic settings:</strong></span>
      <ul>
        <li>Set <strong>Integration type</strong> to <strong>KPN Things</strong>.</li>
        <li><strong>Enable integration</strong> and <strong>Allow create devices or assets</strong> are on by default.</li>
        <li>Click <strong>Next</strong>.</li>
      </ul>
    </li>
    <li>
      <span><strong>Uplink data converter:</strong></span>
      <ul>
        <li>Choose <strong>Create new</strong> — the new converter will contain the necessary code to convert incoming data.</li>
        <li>Click <strong>Next</strong>.</li>
      </ul>
    </li>
    <li>
      <span><strong>Downlink data converter:</strong></span>
      <ul>
        <li>Leave the <strong>Downlink data converter</strong> field empty for now.</li>
        <li>Click <strong>Skip</strong>.</li>
      </ul>
    </li>
    <li>
      <span><strong>Connection:</strong></span>
      <ul>
        <li>Specify your <strong>Base URL</strong>.</li>
        <li>Copy the <strong>HTTP endpoint URL</strong> — you will need it when configuring KPN Things.</li>
      </ul>
    </li>
    <li>Click <strong>Add</strong>.</li>
  </ol>
</Steps>

## KPN Things setup

The following steps walk through configuring the KPN Things Portal to send data to ThingsBoard.
Open the <a href="https://portal.kpnthings.com/" target="_blank">KPN Things Portal</a> and log in to your account.

<Aside type="note">
The KPN Things Portal interface may change in the future.
</Aside>

### Step 2. Add a new device

<Steps>
1. Go to the **Things manager** tab and click **Add new device**.
2. Choose **Device Simulator**, set a name for your device, and click **Add device**.
3. Install and open the **KPN Things Device Simulator** app on your phone. Follow the on-screen instructions to scan the QR code, then click **Add Network Info**.
4. KPN Internet Network is created. Click **Finish**.
</Steps>

### Step 3. Link the device to the flow

<Steps>
1. Go to **Flows** in the left menu and click on the **My first flow** row.
2. Click the **Link Device** button in the **Devices** row.
3. Click the **Link** button next to the device you created earlier.
4. The device is successfully linked to the flow. Click on the **My Device** row to open device details.

### Step 5. Add a new destination

<Steps>
1. Go to **Destinations** in the left menu and click **Add new Destination**.
2. Find the **ThingsBoard** destination type and click **Choose ThingsBoard**.
3. Enter a name for your destination.
4. Paste the **HTTP endpoint URL** from the ThingsBoard integration page into the **HTTP endpoint URL** field.
5. Click the **Generate** button and save the generated token value — you will use it in ThingsBoard to validate the sender.
6. Click **Add ThingsBoard Destination**.
</Steps>

## Update integration with security header

To validate the sender, add the generated token as a security header in the ThingsBoard integration.

<Steps>
1. Go to **Integrations center &#8702; Integrations** and click on the KPN Things integration you created. Click **Edit**.
2. Switch the **Enable security (Headers filter)** toggle to enabled and click **Add** in the **Headers filter** section.
3. Enter `X-Things-Secret` in the **Header** field and paste the generated token into the **Value** field. Click **Apply**.
</Steps>

## Validation

### Send an uplink message

Use the **KPN Things Device Simulator** app on your phone to send a message with data to ThingsBoard. Tap **Send data now** in the app.

<DocImage src="/src/assets/images/user-guide/integrations/kpn/kpn-send-data-now.png" alt="KPN Things Device Simulator app — Send data now button" />

After sending the message, a new device is automatically created in your ThingsBoard instance.

Go to **Devices** in ThingsBoard — you will find the new KPN device with its information and the telemetry received from the device.

<DocImage src="/src/assets/images/user-guide/integrations/kpn/kpn-things-device-1-pe.png" alt="ThingsBoard Devices page — new KPN device with latest telemetry" />

### Check integration events

Go to **Integrations center &#8702; Integrations**, select the KPN Things integration, and open the **Events** tab to see the messages consumed by the integration.
